# Little Ears StoryBox

A gentle bilingual story app for children ages 0-5, with plush-style characters, Chinese and English narration, bedtime mode, sharing, and parent-friendly settings.

## Features

- 12 short stories across bedtime, brave, mealtime, and hug themes
- Chinese and English UI
- Chinese and English MP3 narration
- Character selection with soft plush-style visuals
- Playback controls, progress seeking, and language switching
- Bedtime mode, background music, reading speed, and settings reset
- Share panel with WeChat-friendly copy/share options

## Development

```bash
npm install
npm run dev
```

## Build
