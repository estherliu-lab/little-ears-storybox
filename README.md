## 主要功能

- 中英文界面
- 12 个适合 0-5 岁宝宝的温柔小故事
- 每个故事都有中文和英文 MP3 配音
- 毛绒风小动物角色选择
- 播放、暂停、进度拖动和语言切换
- 睡前模式、背景音乐、朗读速度和清空设置
- 支持微信聊天、朋友圈、系统分享和复制链接的分享面板
- 支持苹果手机和安卓手机添加到桌面
- PWA 结构，方便后续离线和安装优化

## 使用场景

适合睡前听故事、亲子共读、宝宝情绪安抚、吃饭鼓励、勇敢练习，以及日常短时间陪伴。
